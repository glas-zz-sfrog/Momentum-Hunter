# Gateway Product Spec

## Goal
Momentum Hunter should open to a clear two-door choice: Steven Desk or Argus Machine.

## Steven Desk
Steven Desk opens the current human-guided dashboard. It preserves current Daily Workflow, dashboard, watchlist, scoring, readiness, replay, and alert behavior.

## Argus Machine
Argus Machine opens the autonomous console. It is the machine room for planning, simulation, paper trading, broker awareness, and future execution supervision.

## First Screen Requirements
- Two large choices: Steven Desk and Argus Machine.
- Each choice includes a short label describing who is driving the workflow.
- Steven Desk must be the safe path to the current dashboard.
- Argus Machine must be clearly labeled as planning/simulation/supervision until future tasks unlock more.

## Safety Language
The gateway must not imply that Argus Machine is live trading. Use language such as autonomous planning, simulation, paper trading, and supervised execution roadmap.

## Initial Implementation Boundary
ARGUS-A002 may build the shell only. No broker, order, scoring, readiness, replay, or runtime behavior changes are allowed.
